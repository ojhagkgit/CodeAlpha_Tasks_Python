# CodeAlpha Python Programming Internship

This repository contains my submissions for the **Python Programming Internship** at [CodeAlpha](https://www.codealpha.tech). It includes completed implementations for **Task 1: Hangman Game** and **Task 4: Basic Chatbot**.

## 📌 About CodeAlpha

CodeAlpha is a software development company offering internship programs that provide hands-on experience in building real-world applications. This Python track focuses on core programming concepts such as control flow, functions, string processing, and basic I/O.

## ✅ Tasks Completed

| # | Task | Description | Source File |
|---|---|---|---|
| 1 | **Hangman Game** | A text-based Hangman game where the player guesses a randomly selected word one letter at a time, with 6 incorrect guesses allowed and an ASCII-art hangman drawing. | [`CodeAlpha_Hangman.py`](./CodeAlpha_Hangman.py) |
| 4 | **Basic Chatbot** | A rule-based chatbot that replies to greetings, small talk, mood check-ins, and time/date/day queries using keyword matching. | [`CodeAlpha_Chatbot.py`](./CodeAlpha_Chatbot.py) |

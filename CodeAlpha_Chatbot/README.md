# 🤖 Basic Rule-Based Chatbot

A simple rule-based chatbot built in **Python** that responds to predefined user inputs such as greetings, common questions, and exit commands.

This project demonstrates the use of basic Python programming concepts including conditional statements, functions, loops, string manipulation, and input/output.

---

## 📌 Features

- 👋 Responds to greetings
  - Hello
  - Hi
  - Hey
  - Greetings

- 😊 Answers common questions
  - How are you?
  - Who are you?
  - Who built you?
  - Help / What can you do?

- 📅 Provides current
  - Time
  - Date
  - Day

- 💬 Responds to
  - Thank you
  - Positive feelings
  - Negative feelings

- 🚪 Ends the conversation when the user types
  - Bye
  - Goodbye
  - Exit
  - Quit
  - See you

- 🔤 Ignores letter casing
  - HELLO
  - Hello
  - hello

- ✨ Handles punctuation
  - hello!
  - how are you?
  - thanks!
  - bye.

## 💻 Sample Conversation

```
============================================================
            Simple Rule-Based Chatbot

Type 'help' to see what I can do.
Type 'bye', 'exit', or 'quit' to end the chat.
============================================================

You: Hello!
Bot: Hi there! How can I help you today?

You: What can you do?
Bot: I can respond to greetings, questions about myself,
time, day and date, and a few other simple phrases.

You: What's the time?
Bot: The current time is 08:45 PM.

You: Thanks!
Bot: You're welcome!

You: Bye
Bot: Goodbye! Have a great day!
```
